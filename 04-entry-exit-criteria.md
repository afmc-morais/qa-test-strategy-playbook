# Entry & Exit Criteria

## Entry Criteria
- Requirements and acceptance criteria finalized
- Test environment available and stable
- Test data prepared (accounts, permissions, products)
- Build deployed to test environment
- Known issues documented

## Exit Criteria
- Smoke suite passed
- High-risk scenarios validated
- No open Critical defects
- High severity defects have mitigation/decision documented
- Regression completed for impacted modules
- Test report published and communicated
